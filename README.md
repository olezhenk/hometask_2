Проект для управления рецептами, ингредиентами и создания списка покупок.

Общая структура(классы) и функции:
1) Ingredient 
    Методы:
- __init__(name, quantity, unit) – создание ингредиента
- __str__() – формат "название: количество единица"
- __eq__(other) – сравнение по названию и единице (количество не важно)
2) Recipe 
    Методы:
- add_ingredient(ingredient) – добавляет ингредиент (одинаковые складываются)
- scale(ratio) – умножает все ингредиенты на коэффициент
- is_valid_ratio(ratio) – проверяет, что коэффициент > 0
- __len__() – количество ингредиентов
3) DietaryRecipe 
    Методы аналогичны классу Recipe
4) ShoppingList 
    Методы:
- add_recipe(recipe, portions) – добавить рецепт на N порций
- remove_recipe(recipe) – удалить рецепт из списка
- get_list() – вернуть объединённый и отсортированный список ингредиентов
- __add__(other) – объединить два списка

pip install pytest    (если не установлен pytest)
